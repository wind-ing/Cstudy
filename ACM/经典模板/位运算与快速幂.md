# 位运算与快速幂

## 位运算

| 运算符 | 说明     | 运算                   |
| ------ | -------- | ---------------------- |
| &      | 按位与   | 都为1时结果才是1       |
| \|     | 按位或   | 只要有一个1结果就是1   |
| ^      | 按位异或 | 不同为1，相同为0       |
| ~      | 取反     | 取反                   |
| <<     | 左移     | 高位丢弃，低位补0      |
| \>>    | 右移     | 低位丢弃，高位补符号位 |

## 快速幂

### 一般快速幂

作用：快速求$x^n$  
代码：

```C++
#define ll long long
ll quickPow(ll x, ll n)
{
    ll res = 1;
    while(n > 0)
    {
        if(n & 1) res = res * x;
        x = x * x;
        n = (n >> 1);
    }
    return res;
}
```

### 矩阵快速幂

作用：求矩阵$A$的$M$次幂，即$A^M$
代码：

```C++
struct matrix
{
    int m[maxn][maxn];
}ans, res;
matrix mul(matrix a, matrix b)
{
    matrix tmp;
    for(int i = 1; i <= n; ++i)
        for(int j = 1; j <= n; ++j)
            for(int k = 1; k <= n; ++k)
                tmp.m[i][j] += a.m[i][k]*b.m[k][j];
    return tmp;
}
void quickPow(int m)
{
    for(int i = 1; i <= n; ++i)
        for(int j = i; j <= n; ++j)
        {
            if(i == j)
                ans.m[i][j] = 1;
            else
                ans.m[i][j] = 0;
        }
    while(m)
    {
        if(m&1)
            ans = mul(ans, res);
        res = mul(res, res);
        m >>= 1;
    }
}
```
